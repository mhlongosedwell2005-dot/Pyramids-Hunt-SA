# Pyramid Hunt SA

AR treasure hunt game for Namakgale, Limpopo. Built in Sketchware Pro.

### How it works
1. Walk around Namakgale with GPS on
2. Camera scans for virtual pyramids at real GPS spots 
3. Find one → Watch Unity Rewarded Ad → Earn $0.006 to $0.01
4. Unity pays to PayPal at $100 → Withdraw to Capitec

### Setup
1. Open Sketchware Pro
2. Import `SKETCHWARE_CODE.txt` from this repo
3. Replace `YOUR_UNITY_GAME_ID` with your Unity Ads Game ID
4. Build + Run

### Earnings
200 ads/day ≈ R21-R36/day. Unity minimum payout: $100 USD.

### Rules
No self-clicks. No VPNs. Unity bans cheaters.
